# Modern ToDo App

A sleek and efficient ToDo application built using Python and CustomTkinter.


## Features

- Add, update, and delete tasks
- Beautiful CustomTkinter UI
- Voice command integration
- Task persistence (optional: file or database)
- Dark theme


##  Tech Stack

- **Python 3**
- **CustomTkinter**
- **SpeechRecognition**
- **Tkinter**
- **OS, Sys modules**

``` bash
pip install customtkinter
pip install speechrecognition
